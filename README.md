# Проектирование и обучение нейронных сетей 
Бланк билетов 2025-2026

## 1. Понятие о математическом нейроне. Биологический прототип. Искусственный (математический) нейрон. Использование нейронной сети.

Развитие искусственных нейронных сетей (ИНС) исторически вдохновлено биологией, в частности, устройством нервной системы. Основным элементом этой системы является **биологический нейрон** – электрически возбудимая клетка, способная принимать, обрабатывать и передавать информацию с помощью электрохимических сигналов. Он состоит из:
*   **Тела клетки (сомы)**, где происходит суммирование сигналов.
*   **Дендритов** – разветвленных отростков, получающих сигналы от других нейронов.
*   **Аксона** – длинного отростка, передающего сигнал другим нейронам.
*   **Синапсов** – мест контакта между нейронами, где происходит передача сигнала.

Нейрон активируется ("срабатывает"), когда суммарное возбуждение от входящих сигналов превышает определенный порог, после чего он передает импульс по аксону.

Однако искусственные нейронные сети не являются точной копией мозга. Биологическое правдоподобие часто приносится в жертву ради создания эффективных математических моделей, способных решать практические задачи. Биология служит скорее метафорой, чем строгим руководством.

На основе этого прототипа был разработан **искусственный (математический) нейрон** – упрощенная математическая модель, являющаяся базовым вычислительным элементом нейронной сети.

### Структура математического нейрона:
1.  **Входы (X₁, X₂, ..., Xₙ)**: Получают сигналы (числовые значения). Аналоги дендритов.
2.  **Веса (W₁, W₂, ..., Wₙ)**: Каждому входу присвоен вес, который определяет силу (важность) входного сигнала. Аналог силы синаптической связи.
3.  **Суммирующий блок**: Вычисляет взвешенную сумму входных сигналов.
    \[
    NET = \sum_{j=1}^{n} w_j x_j
    \]
    Часто к сумме добавляется **смещение (bias, w₀)**, которое позволяет смещать порог активации, добавляя постоянный вход x₀ = 1.
    \[
    NET = \sum_{j=1}^{n} w_j x_j + w_0
    \]
4.  **Активационная (пороговая) функция (F)**: Применяется к результату суммирования (NET) для получения выходного сигнала нейрона (OUT). Она вносит нелинейность, что является ключевым для мощности нейросетей.
    \[
    OUT = F(NET)
    \]
    Простейшей является **пороговая функция (Хевисайда)**:
    \[
    OUT = \begin{cases}
    1, & \text{если } NET \geq T \\
    0, & \text{если } NET < T
    \end{cases}
    \]
    где T – порог чувствительности.

Таким образом, математический нейрон – это **пороговый элемент с несколькими входами и одним выходом**, преобразующий набор входных сигналов в один выходной согласно взвешенной сумме и нелинейной функции активации.

### Использование нейронных сетей:
Нейронные сети, состоящие из множества соединенных таких нейронов, организованных в слои, используются для решения широкого круга задач, основными из которых являются:
*   **Классификация образов**: Отнесение входных данных (например, изображений, текстов) к одному из заданных классов (распознавание объектов, фильтрация спама, диагностика).
*   **Регрессия**: Прогнозирование непрерывных значений (например, цены акций, температуры).
*   **Кластеризация**: Объединение данных в группы по сходству без предварительно заданных меток.
*   **Аппроксимация функций**: Моделирование сложных нелинейных зависимостей между входными и выходными данными.

Простота математического нейрона позволяет легко его реализовывать и комбинировать в сложные структуры (сети), которые демонстрируют способность к обучению и обобщению на основе данных.
